# SETTribe ERP - Manual Testing Project
Manual testing project for SETTribe ERP including bug reports, test scenarios, and functional testing
## 📌 Overview

This project involves manual testing of the SETTribe ERP system to identify functional and UI defects.

## 🧪 Testing Scope

* Login Module
* Dashboard
* Forms and Input Validation
* Data Filtering

## 🐞 Bugs Found

* Login not working with Enter key
* Invalid input values accepted
* Filter not working with keyboard (requires mouse)
* UI alignment issues

## 🛠 Tools Used

* Microsoft Excel
* Chrome Browser

## 📊 Outcome

* Multiple bugs identified and documented
* Improved understanding of real-time testing
